This is a dummy portfolio website made for the "Build your own portfolio website" workshop of the YoungKWG day. This website is based on the Resume template from Start Bootstrap. To use this template, go to the GitHub repository, click "Use this template", and use [your-github-username].github.io as the repository name. After that, you should be able to find your website on the URL [your-github-username].github.io. You can adapt this template by modifying the index.html file in the git repository.
